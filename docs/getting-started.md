# Getting Started
下面是 `Calculator` 类中的一个加法函数示例：

```csharp
public class Calculator
{
    /// <summary>
    /// 计算两个整数的和。
    /// </summary>
    /// <param name="a">第一个整数</param>
    /// <param name="b">第二个整数</param>
    /// <returns>两数之和</returns>
    public int Add(int a, int b)
    {
        return a + b;
    }
}
```
